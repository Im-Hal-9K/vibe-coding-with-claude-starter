# Vibe Coding with Claude — Starter

Welcome. In this 2-hour session you'll build a small, real, working thing with Claude — and you'll do the building yourself. No experience needed, nothing to install. Everything runs in your browser.

## Start here

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/Im-Hal-9K/vibe-coding-with-claude-starter?quickstart=1)

1. Click the **Open in GitHub Codespaces** button above (sign in with your free GitHub account if asked).
2. Wait a moment while your workspace builds. A code editor opens right in your browser.
3. Open the **Claude panel**, or run `claude` in the terminal. When it asks for an API key, paste the **class key your instructor shows on screen**, then say hello.
4. Tell Claude what you want to build. Use a starter prompt from [`PROMPTS.md`](PROMPTS.md) or your own idea.
5. Run it, look at it, ask Claude to change something. That loop — **describe, look, refine** — is the whole skill.

> The preview opens automatically when your project runs. Your page replaces `index.html`.

## What you'll do today

- Meet Claude and see how plain-language requests become working software.
- Build one small thing you'd actually use (a page, a calculator, a tracker, flashcards, a countdown — see `PROMPTS.md`).
- Make it better, fix what breaks, and learn to keep going on your own.

## If something goes wrong

Tell your instructor. You cannot break anything, and there are no dumb questions. Fallback if Codespaces misbehaves: your instructor will point you to claude.ai with Artifacts.
